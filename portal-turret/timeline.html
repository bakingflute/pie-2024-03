Timeline

A full overview of our project timeline.

Team Logistics

Method of Communication: WhatsApp

Weekly Meeting Times: Wednesdays 3-7PM, Sundays 3-7PM during Sprint 3

Hub of Information and Files: Google Drive, GitHub for code

Sprint 1

Overarching Goals:

  • Set final goals for the project
  • Set team logistics (i.e. method of communication, means of sharing files, meeting times)
  • Start to make purchases and decisions for the means of reaching each overall goal.

Mechanical Achievements

  • Got rack and pinion system fully functional
  • CAD for main body started.

Electrical Achievements

  • Purchase components.
  • Get motors connected to the L293D motor driver and 9V battery.
  • Set up system for moving the rack and pinion system with a motor.
  • Connect a button to test the different features.

Software Achievements

  • Set up SSH and GitHub on the Raspberry Pi.
  • Set up the Raspberry Pi to run Python.
  • Create Python class to play sound.
  • Create Python function to move motors.

Sprint 2

Overarching Goal

    • Get each subteam's simple goals done (chassis, wings moving back and forth, sound, camera)

Mechanical Achievements

  • Made rack and pinion system more stable
  • CAD for main body finished and fabricated

Electrical Achievements

  • Cleaned up wiring using a GPIO breakout board and removing loopy wires.
  • Implemented a sound sensing module with a sound sensor and PCF8591 ADC/DAC.
  • Implemented an accelerometer to measure acceleration and magnetism.

Software Achievements

  • Broke down functionality into more functions and classes.
  • Created a class to play music.
  • Created functions to read sound input and measure acceleration.
  • Added code to take sound input and command motor movement.
  • Implement a rudimentary MP3 player feature.
  • Set up LIRC on the Raspberry Pi.

Sprint 3

Overarching Goals:

  • Get camera functional
  • Communicate more as a team
  • The body needs to hold itself and the electrical components up

Mechanical Achievements

  • CAD for legs done and fully functional (after 2 iterations)
  • Shell materials started - cut out foam pieces to fit around body (proved to not be the move)
  • Servos attached and moving at the ends of the racks

Electrical Achievements

  • Integrate electrical components with mechanical.
  • Add servos to the circuit to control wing rotation.
  • Cleaned up wiring.
  • Added IR receiver to the circuit in order to receive IR codes from a remote control.

Software Achievements

  • Implement IR remote control capabilities.
  • Create cyclable modes, including manual, stimulus detection, music with dancing, and music without dancing.
  • Upgraded the MP3 player feature by adding play/pause, next song, and previous song buttons.
  • Write pose estimation code that can determine which hand is waving.

Final Stretch - Demo Day!

Mechanical Achievements

  • Pivoted to wire and fabric for shell material
  • Body fully built, fully stable

Electrical Achievements

  • Final mechanical-electrical integration.
  • Get all electrical components secured in place.

Software Achievements

  • Move code to a virtual environment.
  • Work on the website.
